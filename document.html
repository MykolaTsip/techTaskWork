Using the program

If you do not have node.JS installed locally to perform the installation on the upcoming link: https://www.youtube.com/watch?v=rHKXBeG4pnY

- Open a command prompt or your own programming environment (IDE)


Step 2 ===> Start the server

- Open the folder: 'bakend'

OR
your own programming environment (IDE) open the terminal ( do not forget to look that we are in the folder "bakend"), prescribe: node app.js

- if you have the inscription: server is work! Congratulations on running the server successfully!
